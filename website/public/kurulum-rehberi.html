SIFIRDAN CANLI YAYINA

MyStore Çark kurulum rehberi

Teknik bilginiz olmadığını varsayarak hazırlandı. Sırayla ilerleyin ve hiçbir adımı atlamayın.

Kısaca: Domaini ekleyin, Çarkı hazırlayın, Entegrasyon bölümündeki kodu kopyalayıp sitenizin </body> etiketinden önce yapıştırın.

1. Başlamadan önce

  • MyStore Çark hesabınız olmalı.
  • Gösterim yapılacak domaini bilmelisiniz; örneğin mynode.com.tr.
  • Site koduna veya Footer Script alanına erişiminiz olmalı.
Erişiminiz yoksa: Embed kodunu siteyi yapan kişiye gönderin ve bütün sayfalarda </body> öncesine eklemesini isteyin.

2. Hesap oluşturma ve giriş

1

MyStore’u açın

Ana sayfadaki “Ücretsiz Başla” butonuna basın.

2

Bilgileri doldurun

Mağaza adı, çalışan e-posta ve en az 8 karakterli şifre girin. Sözleşmeleri onaylayın.

3

Oturum tercihi

“Beni hatırla” tarayıcı kapandığında da oturumu korur. Ortak bilgisayarda seçmeyin.

Şifrenizi unutursanız mail bağlantısı isteyin. Bağlantı bir saat geçerli ve tek kullanımlıktır.

3. Domaini doğru ekleme

Doğru: mynode.com.tr
Yanlış: https://www.mynode.com.tr/anasayfa

https://, www, sayfa yolu ve sondaki çizgiyi yazmayın.

Yanlış domain: Kod sitede olsa bile Çark açılmaz ve 403 hatası oluşur.

4. Çarkı hazırlama

1

Çark Ayarları

Altı dilimin adını, ödülünü, kuponunu ve kazanma ihtimalini düzenleyin.

2

Görünüm

Renkleri, yazıları, boyutu ve dönüş süresini seçin.

3

Canlı Önizleme

Sağ üstteki buton panelden çıkarmadan önizleme kartına götürür.

4

Kaydedin

Değişiklikten sonra Kaydet’e basın.

5. Embed kodunu alma

  1. Sol menüden Entegrasyon açın.
  2. Koyu kod kutusunu bulun.
  3. Kopyala butonuna basın.
  4. “Kopyalandı” mesajını kontrol edin.
<script src="https://cark-backend.onrender.com/dist/cark-widget.js"
        data-store="size-ozel-magaza-slugu"></script>
Panelin verdiği gerçek kodu kullanın. Kodda admin şifresi veya gizli API anahtarı bulunmaz.

6. Kodu siteye ekleme

Normal HTML

index.html veya ortak footer dosyasında </body> öncesine ekleyin.

WordPress

WPCode Footer Script alanını veya child theme footer.php dosyasını kullanın.

React / Vite

Kökteki index.html dosyasının </body> öncesine ekleyin.

Site oluşturucu

Özel Kod, Custom Code, Footer Code veya Body End alanını bulun.

<body>
  <!-- Sitenizin içeriği -->
  <script src="..." data-store="..."></script>
</body>

Kaydedin ve sitenin yayınlama/deploy işlemini yapın.

7. Canlı test

  • Siteyi gizli sekmede açın.
  • Domain adresini kontrol edin.
  • Tetikleme süresini bekleyin.
  • Test bilgileriyle Çarkı çevirin.
  • Sonuç ve kuponu kontrol edin.
  • Katılımcı kaydını panelde bulun.
  • Mobil telefonda deneyin.
  • Test kaydını silin.
Tekrar açılmıyorsa cooldown nedeniyle olabilir; gizli pencere kullanın.

8. Çark görünmüyorsa

Kod canlıya çıktı mı?

Sayfa kaynağında cark-widget.js arayın.

Domain doğru mu?

Panelde yalnızca mynode.com.tr yazmalı.

Kod doğru alanda mı?

Metin alanına değil Footer/Custom Script alanına ekleyin.

Kota veya abonelik sorunu var mı?

Aylık sınır dolduysa ya da abonelik pasifse katılım alınmaz.

Engelleyici açık mı?

Eklentisiz gizli pencerede deneyin.

9. Güvenlik

  • Admin şifresini site koduna yazmayın.
  • İkas Client Secret bilgisini yalnızca panele girin.
  • Ortak bilgisayarda “Beni hatırla” kullanmayın.
  • Yalnızca kontrol ettiğiniz domainleri ekleyin.
  • Gerçek KVKK metninizi hazırlayın.
  • Veremeyeceğiniz ödülleri yazmayın.
